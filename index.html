Welcome to Math Dice.

Math Dice is a simple classroom project based on the board game called Math Dice.

Which languages did I used?

In order to make this web application I used HTML, Javascript, CSS and their respective frameworks known as Bootstrap and Jquery.

Why do this project use so much PHP code?

The reason why I used so many PHP code it's because, as I said, it was a classroom project and the subject was the use of front-end languages.
